Intro
   The file merging process uses the information of the file with extension .assembly_info and merges the parts resulting in the creation of the original file (from which the pieces were created).

Step-by-step merging instructions

  •    To start merging pieces, go to the tab "Merge" and press the button "Search", at the frame titled "Merge Information File", and opens a dialog box from which you can select the name of the information file (wich with extension .assembly_info).
  •    Then press the button "Search", from the frame titled "Folder to store the file" and select the folder where you want to save the file to be created.
  •    Finally, to start the merge process, press the button "Start".
