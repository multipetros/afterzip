Intro
   Through the Settings tab it is possible to customize the application.
In particular, the following can be adapted:

  • The appearance (the theme) of the programs's graphical environment.
  • Display the border of the window (on topics that support this feature).
  • Saves the window dimensions when they are modified by the user.
  • Reset the dimensions to the default.
  • Save the window position when it is modified by the user.
  • Reset the window to the default location.

Themes
   The architecture of the Swing Graphic Environment allows you to customize the appearance of its elements.
So, in addition to Java's default theme, Metal, it is possible to apply modified versions of this topic, but also to scenes that resemble graphic environments, of various operating systems.
Specifically, Sun in the JVM that it distributes has so far included environments:

Motif
The full name of the environment is CDE/Motif, which is designed for UNIX operating systems. Specifically, it simulates OSF/1 Motif 1.2.5, which comes with Sun's Solaris 2.6 operating system.

Gtk
Environment based on the GTK + 2 Open Source Library "The GIMP Toolkit".
In the same library is based the GNOME. One of the most popular integrated support frameworks for the Linux X-Windows system. For this reason the behavior of this subject is similar to GNOME

MS Windows
Microsoft Windows environment. From the versions JVM 1.4.2 + also supports the themable Microsoft Windows XP environment.

Mac OS X
Apple's operating system environment. only supported on machines running Mac OS.
Some of the above themes may not be supported by JVM, which you have on your system.
If you try to apply such a theme, you will receive the relevant notification and will automatically apply the native theme of your system.

Apply a theme to the border
   Swing's 'native' environment, Metal, supports its application in the outline and title bar of the application window. This creates a uniform environment independent of the operating system.
   This technique can not be applied to the issues described above that resemble famous operating system environments.
   If you have applied such a theme to your application selection box and the outline will be inactive.
   In order to apply the theme and the outline, the app's graphical environment must be restarted.

Save resizing
   The application environment, through the layout administrators it uses, allows the user to adjust the window dimensions as needed. With this option turned on, the application window, each time it starts, will have the dimensions the user gave it the last time.

Reset size
   By clicking the 'Reset' button, located next to the box above, the window dimensions are reset to factory default.

Save location change
   With this option turned on, the application window, each time it starts, is displayed in the location where the user left it the last time.

Reset size
   By clicking the 'Reset' button, located next to the selection box above, the window is reset to the default location specified by the manufacturer.
