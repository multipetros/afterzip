Intro
   The spliting process of a file produces pieces of files in the size you specified with .part1, .part2, ... .partN, and a .assembly_info file containing information useful for merging the parts again in a file.

Step-by-step merging instructions

  •    To split a file, go to the Shredding tab and click on the "Search" button in the "File to be Shred" box. A dialog will appear from where you can select the file you want to cut.
  •    Then click on the "Search" button in the box entitled "Common file name of the parts" and in the dialog box that appears, select the path and the common name of the files to be produced. These files will be named selected_name.part1 .... selected_name.partN and selected_name.assembly_info
  •    Then choose the size that you want each piece to have, giving the appropriate value in the text input box in the "Size of each piece" box, or select a ready, standard size from the list that exists.
  •    Finally, to start the merge process, press the button "Start".
